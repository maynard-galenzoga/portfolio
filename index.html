Featured Project

Lightweight HIDPS

Lightweight Host-Based Intrusion Detection and Prevention System (HIDPS) is a web-based security application designed to help resource-constrained servers monitor and neutralize host-level cyber threats quickly, automatically, and efficiently. With real-time host monitoring, it detects unauthorized access, file tampering, and malicious uploads. The system executes automated prevention actions, such as IP blocking and file quarantine, while continually synchronizing threat signatures to ensure adaptive, up-to-date protection.
